# Application-de-recyclage
Implémentation de l'application web EcoFoyer pour la gestion du recyclage domestique  Fonctionnalités : - Page d'accueil avec présentation des fonctionnalités principales - Programme hebdomadaire de recyclage avec calendrier interactif - Interface de sélection et validation des déchets recyclés - Historique de validation avec suivi des progrès
